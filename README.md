### About
This repository contains Windows 7 game pack for Linux. Original executables are patched to run through Wine.

### Requirements
The only requirement is Wine. Install it from your distribution repository.

### Installation
```
# Clone this repository
git clone https://github.com/xSpaceC4t/win7games-on-wine.git ~/win7games-on-wine

# Set PATH variable in ~/.bashrc (optional)
echo "export PATH=$PATH:$HOME/win7games-on-wine" >> ~/.bashrc
```
